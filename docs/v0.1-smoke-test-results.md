## Issues found

- `tmux` not installed on myflix — server started via `nohup` instead. Survives SSH disconnection but does not survive a server reboot. Follow-up: create a systemd user service (see Follow-ups below).
- Port 8000 was not open in myflix firewall — opened permanently via `sudo firewall-cmd --add-port=8000/tcp --permanent && firewall-cmd --reload`.
